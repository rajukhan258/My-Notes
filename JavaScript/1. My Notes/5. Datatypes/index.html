Types of Data in JavaScript

There are two types of data in JavaScript:

1. Primitive data type

Primitives are the simplest and most basic data types in JavaScript. They include:

  • Numbers (e.g. 10, 3.14) Let x= 25;
  • Strings (e.g. "hello", 'world') Let= "Hello"
  • Booleans (e.g. true, false) Var x= "true"
  • BigInt (e.g. "567") Let x= BigInt("567")
  • Symbol (e.g. "I am a symbol")
  • Null (a special value that represents an absence of value)
  • Undefined (a special value that represents an uninitialized variable)

2. Non-primitive Data Type

Objects are non-primitive data types in JavaScript. Objects are data that have methods and properties.

The object data type can contain:

  • An object Let x= {first:"Jane", last:""Doe};
  • An array Let x= ["HTML", "CSS", ""JS];
  • A date const date = new Date("2022-03-25");
