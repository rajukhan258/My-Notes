Math Object

Math is a built-in object which has many properties and methods that are used to deal with mathematical constants and functions like PI, sin, cos, etc.

  • Math Object Properties
  • Math Object methods

Math Object Properties (Constants)

The syntax for any Math property is : Math.property. JavaScript provides 8 mathematical constants that can be accessed as Math properties:

  1. Math.PI: Returns the value of mathematical constant PI
  2. Math.E: Returns Euler's constant (base of natural logarithm)
  3. Math.LN2: Returns the value of the natural logarithm of 2 (approx 0.693)
  4. Math.LN10Returns the value of the natural logarithm of 10 (approx 2.302)
  1. Math.LOG2E: Returns the value of base 2 logarithm of e (approx 1.443)
  2. Math.LOG10E: Returns the value of base 10 logarithm of e (approx 0.434)
  3. Math.SQRT1_2: Returns the value of the square root of 1/2 (approx 0.707)
  4. Math.SQRT2: Returns the value of the square root of 2 (approx 1.414)

1. Math.PI

PI is a static property of the Math object. It returns the value of mathematical PI (π).

It's a commonly used constant in mathematics and programming for calculations involving circles and spheres.

Example

Unlike Array, Object and date Objects, it doesn't have constructors.

1. Math.round()

Math.round() is a method in JavaScript's Math object used to round a number to the nearest integer. The rounding is based on standard mathematical rounding rules:

  • If the decimal part of the number is 0.5 or greater, Math.round() will round the number up to the next integer.
  • If the decimal part is less than 0.5, Math.round() will round the number down to the previous integer.

NOTE:

  • Math.round() does not modify the original number; it returns a new integer value based on the rounding rules.
  • If the number is exactly halfway between two integers (e.g., 2.5), Math.round() will round to the nearest even number (this is known as "round half to even" or "bankers' rounding"). For example, Math.round(2.5) results in 2, and Math.round(3.5) results in 4.

Example

<script>
    // Perform rounding operations on different numbers
    let roundOff1 = Math.round(3.2);
    let roundOff2 = Math.round(3.8);
    let roundOff3 = Math.round(-3.2);
    let roundOff4 = Math.round(-3.8);
    let roundOff5 = Math.round(2.5);
    let roundOff6 = Math.round(3.5);

    // Display the results in HTML element
    document.getElementById('round-output-1').innerHTML = `${roundOff1} <br> ${roundOff2} <br>${roundOff3} <br>${roundOff4} <br>${roundOff5} <br>${roundOff6}`;
</script>

Math.floor() is a method in JavaScript's Math object used to round a number down to the nearest integer that is less than or equal to the given number. It essentially truncates the decimal part of a number, returning the largest integer less than or equal to the number itself.

It doesn't modify the original number; it returns a new integer value.

10. Math.pow()

Math.pow() is a method in JavaScript's Math object used to raise a base number to an exponent power.

Math.pow() takes two arguments: base (the base number) and exponent (the power to which the base is raised). Both base and exponent can be positive, negative, or zero.

If the exponent is 0, the result will always be 1, regardless of the base.

If the exponent is a fraction or decimal, Math.pow() can be used to calculate roots or fractional powers. For example, Math.pow(16, 0.5) calculates the square root of 16, resulting in 4.

In modern JavaScript, the exponentiation operator (**) is also available for the same purpose and often used for its simplicity. For instance, 2 ** 3 is equivalent to Math.pow(2, 3) and returns 8.

Example

<script>
    // Calculate powers using Math.pow() and the exponentiation operator
    let pow1 = Math.pow(2, 3);
    let pow2 = Math.pow(2, -3);
    let pow3 = Math.pow(16, 0.5);
    let pow4 = Math.pow(10, 0);
    let pow5 = 2 ** 3;

    // Display the results in an HTML element
    document.getElementById('pow-output-1').innerHTML = `${pow1}<br>${pow2}<br>${pow3}<br>${pow4}<br>${pow5}`
</script>

13. Math.log()

Math.log() is a method in JavaScript's Math object used to calculate the natural logarithm (base e) of a given number.

Logarithms are the inverse operation to exponentiation.

How many times must we multiply Math.E to get 10?

In the expression log(10), the function calculates the natural logarithm. This asks: "To what power must e be raised to give 10?" The answer is approximately 2.30259, as e2.30259 ≈ 10.

For positive numbers, it returns the natural logarithm of the given value.

  • Return NaN (Not a Number) for negative numbers, as there's no valid logarithm for negative values.
  • -Infinity for zero, as the logarithm of zero to any base is negative infinity

16. Math.sin()

Math.sin() is a JavaScript function that calculates the sine of an angle given in radians.

Math.sin() expects input in radians. To convert degrees to radians, use

let degToRedian = degrees * Math.PI/180;

For an angle of π/6 radians (which represents 30 degrees), sin(π/6) returns 0.5 because the sine of π/6 radians is 0.5.

The Math.sin() method returns a number between -1 and 1. NaN if the parameter is not numeric.

For a sine value of 0.5, asin(0.5) returns approximately 0.5236 radians because the arcsine of 0.5 is approximately π/6 radians.

  • Math.asin() expects input values between -1 and 1, representing valid sine values.
  • The function returns values in radians within the range [-π/2, π/2]. NaN if the parameter is not numeric or outside the range of -1 to 1.

To convert an angle in radians to degrees, the formula is:

21. Math.atan()

Math.atan() is a JavaScript function that calculates the arctangent (inverse tangent) of a number, returning an angle between -PI/2 and PI/2 radians.

For a tangent value of 1, atan(1) returns approximately 0.7854 radians because the arctangent of 1 is approximately π/4 radians.

  • Math.atan() expects input values ranging from negative infinity to positive infinity, representing valid tangent values.
  • The function returns values in radians within the range [-π/2, π/2]. NaN if the number is empty or not numeric.

22. Math.atan2()

The Math.atan2() method returns the arctangent of the quotient of its arguments, as a numeric value between PI and -PI radians.

Note: With atan2(), the y coordinate is passed as the first argument and the x coordinate is passed as the second argument.

For x = 1 and y = 1, atan2(1, 1) returns approximately 0.7854 radians because the arctangent of the quotient y/x (1/1) is approximately π/4 radians.

  • It takes two arguments: Math.atan2(y, x)
  • The function returns values in radians within the range [-π, π], representing the angle from the positive x-axis to the point (x, y).
  • Returns NaN if the values are empty or not numeric.
