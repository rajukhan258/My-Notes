String Methods

List of String Methods:

  1. str.toLowerCase(): converts all characters in a string to lowercase, creating a new string without altering the original.
  2. str.toUpperCase(): converts all characters in a string to uppercase, creating a new string without altering the original.
  3. str.includes(): checks if a string contains a specified substring, returning true or false accordingly.
  4. str.startsWith(): checks if a string starts with a specified substring, returning true or false accordingly.
  5. str.endsWith(): checks if a string ends with a specified substring, returning true or false accordingly.
  6. str.search(): searches for a specified substring within a string and returns the index of the first occurrence, or -1 if not found.
  7. str.indexOf(): finds the index of the first occurrence of a specified substring within a string, returning the index or -1 if not found.
  8. str.lastIndexOf(): finds the index of the last occurrence of a specified substring within a string, returning the index or -1 if not found.
  9. str.match(): retrieves all matches when matching a string against a regular expression, returning an array of matches or null if no matches are found.
  10. str.replace(): finds a specified substring (or pattern) within a string and replaces it with another substring, returning the modified string.
  11. str.trim(): removes leading and trailing whitespace from a string, returning a new string with the whitespace removed without altering the original string.
  1. str.charAt(): retrieves and returns a specific character from a string based on its index position, starting from 0.
  2. str.charCodeAt(): fetches the Unicode value of a character at a particular index within a string.
  3. string.fromCharCode(): generates a string from one or more Unicode values by representing the characters corresponding to those Unicode values.
  4. str.concat(): combines strings together, creating a new string with the content of the original strings appended in the order they are provided.
  5. str.split(): divides a string into an array of substrings based on a specified separator and returns a new array.
  6. str.repeat(): duplicates a string by the number of times specified, creating and returning a new string with the repeated content.
  7. str.slice(): extracts a section of a string and returns a new string without altering the original, defined by the specified start and end indices.
  8. str.substr(): extracts a substring of a specified length starting from a specified index within a string, returning a new string without changing the original.
  9. str.substring(): retrieves a substring between specified indices within a string, providing a new string without altering the original content.
  10. str.toString(): Calling .toString() on a string in JavaScript returns the string itself; for non-string types (like numbers, booleans, or objects), it generates and returns a new string representation of that value.
  11. str.valueOf(): retrieves and returns the primitive value of a String object, which is the string itself.

1. str.toLowerCase()

str.toLowerCase() is a method used to convert all the characters in a string to lowercase.

It returns a new string with all the characters converted to their lowercase equivalents.

Example

<script>
    // Declare a variable 'myString' and assign the string "Hello, World!" to it
    let myString = "Hello, World!";

    // Apply the toLowerCase() method to 'myString' to convert it to lowercase
    let lowerCaseString = myString.toLowerCase();

    // Set the inner HTML of an element with the ID 'toLowerCase-output-1' to the lowercase string
    document.getElementById("toLowerCase-output-1").innerHTML = lowerCaseString;
</script>

3. str.includes()

str.includes() is a method used to check if a string contains another specified string. It returns true if the string contains the specified substring, otherwise false.

Example

<script>
    // Initial string
    let myString = "Hello World";

    // Check if 'myString' includes "Hello"
    let checkHello = myString.includes("Hello");

    // Check if 'myString' includes "Goodbye"
    let checkGoodbye = myString.includes("Goodbye");

    // Display results in HTML elements
    document.getElementById("includes-output-1").innerHTML = `includes "Hello": ${checkHello}`;
    document.getElementById("includes-output-2").innerHTML = `includes "Goodbye": ${checkGoodbye}`;
</script>

4. str.startsWith()

str.startsWith() is a method used to check if a string starts with a specified substring.

It returns true if the string begins with the specified characters, otherwise false.

Example

<script>
    // Define the string
    let myString = "Hello World!";

    // Check if 'myString' starts with "He" and "Hi"
    let startsWithHello = myString.startsWith("He");
    let startsWithHi = myString.startsWith("Hi");

    // Display the results in HTML elements
    document.getElementById("startsWith-output-1").innerHTML = `Starts With "Hello": ${startsWithHello}`;
    document.getElementById("startsWith-output-2").innerHTML = `Starts with "Hi": ${startsWithHi}`;
</script>

5. str.endsWith()

str.endsWith() is a method used to check if a string ends with a specified substring.

It returns true if the string ends with the specified characters, otherwise false.

Example

<script>
    // Define the string
    let myString = "Hello, World!";

    // Check if 'myString' ends with "rld!" and "Planet!"
    let endsWithWorld = myString.endsWith("rld!");
    let endsWithPlanet = myString.endsWith("Planet!");

    // Display the results in HTML elements
    document.getElementById("endsWith-output-1").innerHTML = endsWithWorld;
    document.getElementById("endsWith-output-2").innerHTML = endsWithPlanet;
</script>

6. str.search()

str.search() is a method used to search for a specified substring within a string.

It returns the index of the first occurrence of the substring within the string. If the substring is not found, it returns -1.

indexOf() is simpler and primarily used for basic substring searches, while search() is more flexible due to its ability to use regular expressions for pattern-based searches.

Example

<script>
    // Define the string
    let myString = "Hello, World!";

    // Search for "World" and "Universe" within 'myString'
    let searchResult1 = myString.search("World");
    let searchResult2 = myString.search("Universe");

    // Display the search results in HTML elements
    document.getElementById("search-output-1").innerHTML = `Index of world: ${searchResult1}`;
    document.getElementById("search-output-2").innerHTML = `Index of Universe: ${searchResult2} (not found)`;
</script>

7. str.indexOf()

str.indexOf() is a method used to find the index of the first occurrence of a specified substring within a string.

It returns the index of the first occurrence of the substring or -1 if the substring is not found.

indexOf() is simpler and primarily used for basic substring searches, while search() is more flexible due to its ability to use regular expressions for pattern-based searches.

Example

<script>
    // Define the string
    let myString = "Hello, World!";

    // Find the index of "World" and "Universe" within 'myString'
    let indexWorld = myString.indexOf("World");
    let indexUniverse = myString.indexOf("Universe");
    
    // Display the indices in HTML elements
    document.getElementById("indexOf-output-1").innerHTML = indexWorld;
    document.getElementById("indexOf-output-2").innerHTML = indexUniverse;
</script>

8. str.lastIndexOf()

str.lastIndexOf() is a method used to find the index of the last occurrence of a specified substring within a string.

It returns the index of the last occurrence of the substring or -1 if the substring is not found.

Example

<script>
    let myString = "Hello, World!";

    //Find the last occurrence of each character
    let lastIndexL = myString.lastIndexOf("l");
    let lastIndexM = myString.lastIndexOf("M");

    // Display the indices in HTML elements
    document.getElementById("lastIndexOf-output-1").innerHTML = `last Index Of "l": ${lastIndexL}`;
    document.getElementById("lastIndexOf-output-2").innerHTML = `last Index Of "M": ${lastIndexM}`;
</script>

9. str.match()

str.match() is a method used to retrieve the matches when matching a string against a regular expression.

It returns an array containing the matches found or null if no matches are found.

Example

<script>
    let myString = "The rain in Spain falls mainly in the plain";

    // Using match() with a regular expression to find all occurrences of 'ain'
    let matchResult = myString.match(/ain/g);

    // Display the match results in HTML elements
    document.getElementById("match-output-1").innerHTML = `[${matchResult.join(", ")}]`;
</script>

10. str.replace()

str.replace() is a method used to replace a specified substring (or a pattern defined by a regular expression) within a string with another substring.

It returns a new string with the specified substring (or a pattern defined by a regular expression) replaced by another substring. It does not modify the original string.

Example

<script>
    let myString = "Hello, World!";
    
    // Replace "World" with "Guys" in 'myString'
    let replacedString = myString.replace("World", "Guys");

    // Replace all occurrences of "ello" with "i" using a regular expression in 'myString'
    let replaceString2 = myString.replace(/ello/g, "i");

    // Display the replaced strings in HTML elements
    document.getElementById("replace-output-1").innerHTML = replacedString;
    document.getElementById("replace-output-2").innerHTML = replaceString2;
</script>

11. str.trim()

str.trim() is a method used to remove whitespace characters (spaces, tabs, newlines, etc.) from both the beginning and the end of a string.

It returns a new string with the leading and trailing whitespace removed.

Example

<script>
    // Define the string with leading and trailing spaces
    let myString = "   Hello, World!   ";

    // Remove leading and trailing spaces from 'myString'
    let trimmedString = myString.trim();

    // Display the trimmed string in an HTML element
    document.getElementById("trim-output-1").innerHTML = trimmedString;
</script>

14. String.fromCharCode()

String.fromCharCode() is a method in JavaScript used to create a string from a sequence of Unicode values.

It takes one or more Unicode values and returns a string representing the characters corresponding to those Unicode values.

Example

<script>
    // Creates a string from Unicode value 65 (A)
    let char1 = String.fromCharCode(65);

    // Creates a string from Unicode values for "Hello"
    let char2 = String.fromCharCode(72, 101, 108, 108, 111);

    // Display output in an HTML element
    document.getElementById("fromCharCode-output-1").innerHTML = char1;
    document.getElementById("fromCharCode-output-2").innerHTML = char2;
</script>

15. str.concat()

str.concat() is a method in JavaScript used to concatenate (join together) two or more strings and return a new string that contains the combined text of the original strings.

Example

<script>
    let str1 = "Hello ";
    let str2 = "Guys! ";
    let str3 = "How are you?";

    let concat1 = str1.concat(str2);    // Concatenates str1 and str2
    let concat2 = str1.concat(str2, str3);  // Concatenates str1, str2 and str3

    // Display output in an HTML element
    document.getElementById("concat-output-1").innerHTML = concat1;
    document.getElementById("concat-output-2").innerHTML = concat2;
</script>

18. str.slice()

str.slice() is a method in JavaScript used to extract a portion of a string and return it as a new string without modifying the original string.

  • the ending index (exclusive)

Example

21. str.toString()

str.toString() is a method in JavaScript used to convert a string to its string representation. While strings are already represented as strings, this method is available for consistency with other data types that have a toString() method.

In JavaScript, calling .toString() on a string doesn't create a new string; it simply returns the string itself. However, for non-string types like numbers, booleans, or objects, invoking .toString() generates and returns a new string representation of that value.

Example

<script>
    // Assigns a numeric value to the variable num
    let num = 42;

    // Converts the numeric value to its string representation
    let stringRepresentation = num.toString();

    // Displays the string representation in an HTML element
    document.getElementById("toString-output-1").innerHTML = stringRepresentation;
</script>

22. str.valueOf()

str.valueOf() is a method in JavaScript used to retrieve the primitive value of a string object. This method returns the primitive string value of a String object, which is the string itself.

Example

<script>
    let strObject = new String("Hello");

    // Retrieves the primitive value of the String object
    let primitiveValue = strObject.valueOf();

    // Displays the string representation in an HTML element
    document.getElementById("valueOf-output-1").innerHTML = primitiveValue;
</script>
