JS DOM Get Set Attribute

When the browsers load HTML and generate DOM object from it, then most standard HTML attributes automatically become properties of the DOM objects.

For example, if the tag is '<p title="paragraph title">, then the DOM object has 'thisParagraph.title = "paragraph title"`.

Every attribute of an element does not become DOM object property, only the standard attribute of the element becomes DOM object property.

Javascript provides some methods to access and manage such attributes. We can use 4 different javascript method to handle attributes within the DOM.

  1. hasAttribute()
  2. getAttribute()
  3. setAttribute()
  4. removeAttribute()

1. hasAttribute()

The element.hasAttribute() method is a javascript method to check whether a specific element has a specific attribute or not. It returns a boolean value.

"hasAttribute" method is used to check whether an attribute exist or not.

This is a paragraph.

<div class="d-none test">
	<h2>"hasAttribute" method is used to check whether an attribute exist or not.</h2>
	<p name="my paragraph">This is a paragraph.</p>
</div>

Example

<script>		
	document.querySelector('#hasattrbtn').addEventListener("click", () => {
	const element = document.querySelector('.test p');
	if (element.hasAttribute('name')) {
		alert("Yes 'name' attribute exist!");
	}
	else {
		alert("No 'name' attribute doesn't exist!");
	}
});
</script>

2. getAttribute()

The getAttribute() method is used to get the attribute value from any DOM element. This method returns the value of the attribute specified if the attribute doesn't exist then value returned is null or "" (empty string).

"hasAttribute" method is used to check whether an attribute exist or not.

This is a paragraph.

<div class="d-none test2">
	<h2>"hasAttribute" method is used to check whether an attribute exist or not.</h2>
	<p class="my paragraph">This is a paragraph.</p>
</div>

Example

<script>
document.querySelector('#getattrbtn').addEventListener('click', () => {
	const target = document.querySelector('.test2 p');
	const classValue = target.getAttribute('class');
	if (classValue) {
		document.querySelector('#output-getattr').innerText = classValue;
	}
	else {
		document.querySelector('#output-getattr').innerText = "not Found!";
	}
});
</script>

3. setAttribute()

The setAttribute() method is used to set new attribute and its value to the DOM element. If the attribute already exists then the value of the attribute is updated with the new value.

Syntax

element.getAttribute(name, value);

This method takes 2 argument name and value.

"hasAttribute" method is used to check whether an attribute exist or not.

<div class="d-none test3">
