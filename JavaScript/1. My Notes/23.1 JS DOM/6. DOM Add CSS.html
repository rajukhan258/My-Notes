Add DOM CSS

We can use javascript to directly set a style on an element and you can also use it to add or remove classes from the elements which will add or remove a set of CSS style to the element.

  1. Directly setting CSS style on element
  2. Adding or removing class from the element
<script>
	const target = document.querySelector('#sample1');
	target.style.backgroundColor = '#363636';
	target.style.color = '#ffffff';
</script>

Special case: The CSS property float is used as cssFloat when using with javascript because the word float is a reserved word in javascript.

2. Adding or removing class from the element

In JavaScript, we have className and classList properties to handle classes.

  • className: It returns all the classes present in the element in form of string
  • classList: It returns DOMTokenList, which contains various methods to handle classes in an element

className

The className property is an older approach that sets or retrieves the value of the class attribute of an HTML element as a single string. While the classList property provides a more modern and powerful way to manipulate classes

This is a div element.

<div id="myDiv" class="d-none">
	<p>This is a div element.</p>
</div>

Example

<script>		
	document.querySelector('#classnamebtn').addEventListener('click', () => {
	const target = document.querySelector('#myDiv p');
	target.className += ' newClass-added';
	alert(myDiv.innerHTML);
});
</script>

classList

The classList property returns a collection of all classes of an element, allowing you to manipulate them easily. Here are few methods to modify classes:

  • classList.add()
  • classList.toggle()
  • classList.remove()
  • classList.replace()

a. classList.add()

To add a class value to an element use add method on an element via classList property of the element.

Single or multiple classes can be added to the element at once. To add multiple classes provide a class name in the add method separated by a comma.

Adding a single class to the element.

element.classList.add('size');

Adding multiple classes to the element.

element.classList.add('padding', 'color');

Here is a working example:

This is a div element.

<div id="myDiv" class="d-none">
	<p>This is a div element.</p>
</div>

Example

document.querySelector('#classlist1').addEventListener('click', () => {
	const target2 = document.querySelector('#Div2 p');
	target2.classList.add('size');
	alert(Div2.innerHTML);
});

document.querySelector('#classlist2').addEventListener('click', () => {
	const target2 = document.querySelector('#Div2 p');
	target2.classList.add('size', 'color', 'border');
	alert(Div2.innerHTML);
});

b. toggle class in javascript

The toggle method of classList property toggle a class value from the element means if the class is present in the element then it removes the class and if the class is not present in the element then it adds the class to the element.

Example

document.querySelector('#togglebtn').addEventListener('click', () => {
	const target3 = document.querySelector('#toggle pre');
	target3.classList.toggle('bg-light');
});

c. remove class in javascript

To remove a class from an element use remove method of classList property.

Removing single class:

element.classList.remove('size');

Removing multiple class:

element.classList.remove('padding', 'color');

This is a div element.

<div id="remove" class="d-none">
	<p class="size border font">This is a div element.</p>
</div>

Example

document.querySelector('#removebtn1').addEventListener('click', () => {
	const target2 = document.querySelector('#remove p');
	target2.classList.remove('size');
	alert(remove.innerHTML);
});

document.querySelector('#removebtn2').addEventListener('click', () => {
	const target2 = document.querySelector('#remove p');
	target2.classList.remove('size', 'border', 'font');
	alert(remove.innerHTML);
});

d. replace class in javascript

To replace a class with another class in an element use replace() method with classList API.

The replace method takes two arguments, the first argument is old class and the second argument is a new class.

Replacing a class with another class:

element.classList.replace('blue', 'red');

This is a div element.

<div id="replace" class="d-none">
	<p class="size">This is a div element.</p>
</div>

Example

document.querySelector('#replacebtn').addEventListener('click', () => {
	const target5 = document.querySelector('#replace p');
	target5.classList.replace('size', 'border');
	alert(replace.innerHTML);
});

JS DOM Animation

Create an Animation Container

All animations should be relative to a container element.

<div id ="container">
  <div id ="animate">My animation will go here</div>
</div>

Style the Elements

The container element should be created with style = "position: relative".

The animation element should be created with style = 'position: absolute'.

#container {
  width: 400px;
  height: 400px;
  position: relative;
  background: yellow;
}
#animate {
  width: 50px;
  height: 50px;
  position: absolute;
  background: red;
}

Animation Code

JavaScript animations are done by programming gradual changes in an element's style. The changes are called by a timer. When the timer interval is small, the animation looks continuous.

id = setInterval(frame, 5);

function frame() {
  if (/* test for finished */) {
    clearInterval(id);
  } else {
    /* code to change the element style */ 
  }
}

Example

function myMove(){
	let id = null;
	const elem = document.getElementById("animate");
	let pos = 0;
	id = setInterval(frame, 5);
	function frame(){
		if(pos == 250){
			clearInterval(id);
		} else{
			pos++;
			elem.style.top = pos + "px";
			elem.style.left = pos + "px";
		}
	}
}
