DOM Document

The Document is an interface that represents the web pages. It serves as an entry point into the content of web pages i.e DOM Tree.

The document is the root node of the DOM tree. It is the top-level container for all other nodes in the DOM tree.

The document includes everything about the webpage both which is visible and which is not visible on the screen. In other words, the document is the owner of all the node objects in the webpage.

List Of Document Object Properties

Property Description
document Represents the entire HTML document and serves as the entry point to the DOM
document.all An outdated property that once represented all elements in the document (not recommended for use).
document.documentElement Refers to the root element () of the document.
document.head References the element of the document.
document.title Retrieves or sets the title of the document shown in the browser's title bar or tab.
document.body Refers to the <body> element of the document.
document.images Represents a collection of all elements in the document.
document.anchors Provides access to a collection of all <a> elements with a 'name' attribute.
document.links Refers to a collection of all <a> elements with an 'href' attribute.
document.forms Represents a collection of all <form> elements in the document.
document.doctype Represents the DocumentType node of the document
document.URL Retrieves the complete URL of the document
document.baseURI Provides the base URL of the document
document.domain Represents the domain name of the server from which the document originated (may be restricted by same-origin policy).
Document.doctypeURI It returns the location (URL) of the current document
Document.cookie It returns a semicolon-separated list of the cookies for the document
Document.activeElement It returns the element cursor is focusing to in the document
Document.characterSet It returns character encoding for the document
Document.charset It returns character encoding for the document
