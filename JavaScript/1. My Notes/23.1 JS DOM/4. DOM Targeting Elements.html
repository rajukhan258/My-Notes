Methods for Targeting DOM Objects

Methods for Targeting DOM Objects:

  1. getElementById(): Targets an element with a specific ID.

    2. Copy the Code and Paste it in console to check output

    // Get element by ID
const targerId = document.getElementById('sampleId-1');
console.log(targerId);
  2. getElementsByClassName(): Selects multiple elements by their class name.

    3. Copy the Code and Paste it in console to check output

    // Get element by ClassName
const targetClass = document.getElementsByClassName('text-danger');
console.log(targetClass);
  3. getElementsByTagName(): Grabs elements by their tag name.

    4. Copy the Code and Paste it in console to check output

    // Get element by TagName
const targetTagName = document.getElementsByTagName('p');
console.log(targetTagName);
  4. querySelector(css selector): Retrieves the first element that matches the specified CSS selector.

    5. Copy the Code and Paste it in console to check output

    // Using querySelector
const selectTag = document.querySelector('p');
const selectClass = document.querySelector('.text-danger');
const selectId = document.querySelector('#sampleId-1');
const AdvSelect = document.querySelector('#DOMDefinition p');

console.log(selectTag);
console.log(selectClass);
console.log(selectId);
console.log(AdvSelect);
  5. querySelectorAll(css selector):Returns a NodeList of all elements matching a specified CSS selector.

    6. Copy the Code and Paste it in console to check output

    // Using querySelectorAll
const selectAllTag = document.querySelectorAll('p');
const selectAllClass = document.querySelectorAll('.text-danger');
const selectAllId = document.querySelectorAll('#sampleId-1');
const AdvAllSelect = document.querySelectorAll('#DOMDefinition p');

console.log(selectAllTag);
console.log(selectAllClass);
console.log(selectAllId);
console.log(AdvAllSelect);
