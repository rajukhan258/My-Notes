DOM (Document Object Model)

The W3C Document Object Model (DOM) is a platform and language-neutral interface that allows programs and scripts to dynamically access and update the content, structure, and style of a document.

The W3C DOM standard is separated into 3 different parts:

  • Core DOM - standard model for all document types
  • XML DOM - standard model for XML documents
  • HTML DOM - standard model for HTML documents

HTML DOM:

The DOM stands for Document Object Model, it is an interface that allows developers to interact and control the Elements of the web. It represents an HTML document as a tree of Nodes. It provides functions that allow you to add, remove, and modify parts of the document effectively.

Img: The HTML DOM Tree of Objects

Nodes & It's Types

Nodes are individual components or entities that collectively construct the hierarchical structure of an HTML document.

The DOM comprises different types of nodes:

  • Element Nodes: Element nodes are themselves called objects in DOM. It comprises most of the DOM which holds other data like text and attributes. Example p, h1, h2, ul, li etc. The element nodes may have other elements nested inside it which are known as children of that element.
  • Attribute Nodes: Attribute nodes are attached to the element node and add extra information to the element. Example title attribute, class attribute etc.
  • Text Node: ext nodes are contents of the elements and have no children like element nodes.

Each node in the DOM tree is identified by a node type. JavaScript uses integer numbers to determine the node types. Each type of node has a different way to access.

Constant Value Description
node.ELEMENT_NODE 1 An Element node like <p> or <img>
node.ATTRIBUTE_NODE 2 An attribute of an element
node.TEXT_NODE 3 The text inside an element
node.COMMENT_NODE 8 Any comment node like <!--commment-->
node.DOCUMENT_NODE 9 A document node

A node has two important properties:

  • nodeName
  • nodeValue

DOM Get Methods:

  1. innerText: Represents the visible text content of an element.

    2. Example

    <div id="myDiv">Hello, World!</div>
<script>
  let textContent = document.getElementById('myDiv').innerText;
  console.log(textContent); // Output: Hello, World!
</script>
  2. innerHTML: Represents the HTML content within an element.

    3. Example

    <div id="myDiv"><strong>Hello, World!</strong></div>
<script>
  let htmlContent = document.getElementById('myDiv').innerHTML;
  console.log(htmlContent); // Output: <strong>Hello, World!</strong>
</script>
  3. getAttribute: Retrieves the value of the specified attribute of an element.

    4. Eample

    <a id="myLink" href="https://example.com">Click Here</a>
<script>
  let hrefValue = document.getElementById('myLink').getAttribute('href');
  console.log(hrefValue); // Output: https://example.com
</script>
  4. getAttributeNode: Returns the specified attribute node of an element.

    5. Example

    <input id="myInput" type="text" value="Hello">
<script>
  let typeAttribute = document.getElementById('myInput').getAttributeNode('type');
  console.log(typeAttribute); // Output: Attr { name: "type", value: "text", ... }
	let name = document.getElementById('myInput').getAttributeNode('type').name;
	let value = document.getElementById('myInput').getAttributeNode('type').value;
</script>
  5. Attributes: Represents a collection of all attributes of an element.

    6. Example

    <img id="myImage" src="image.jpg" alt="My Image">
<script>
  let attributes = document.getElementById('myImage').attributes;
  console.log(attributes); // Output: NamedNodeMap { 0: src, 1: alt, 2: id, ... }
	let name = document.getElementById('myImage').attributes[1].name;
	let value = document.getElementById('myImage').attributes[2].value;
</script>

DOM Set Methods:

  1. innerText: Represents the text content of an element, excluding any child elements and their content.

    2. Example

    <div id="myDiv">
  <h1><strong>Hello World</strong></h1>
</div>
<script>
	document.getElementById('myDiv').innerText = "Hi!"
  let textContent = document.getElementById('myDiv').innerText;
  console.log(textContent); // Output: "Hi!"
</script>
  2. innerHTML: Represents the HTML content within an element, including any child elements and their content.

    3. Example

    <div id="myDiv">
  <strong>Hello, <em>World</em>!</strong>
</div>
<script>
document.getElementById('myDiv').innerHTML = "<h1>Hello World</h1>";
  let htmlContent = document.getElementById('myDiv').innerHTML;
  console.log(htmlContent); // Output: "<h1>Hello World</h1>"
</script>
  3. setAttribute: Sets the value of the specified attribute on the element.

    4. Example

    <div id="myDiv">Hello, World!</div>
<script>
  document.getElementById('myDiv').setAttribute('class', 'highlight');
  document.getElementById('myDiv').setAttribute('style', 'background-color: red; color: white;');
 	console.log(document.getElementById('myDiv'));
 // Output: <div id="myDiv" style="background-color: red; color: white;" class="highlight">Hello, World!</div>
</script>
  4. Attribute: Represents a specific attribute of an HTML element.

    5. Example

    <div id="myDiv" style="background-color: red; color: white;" class="highlight">Hello, World!</div>
<script>
  let a = document.getElementById('myDiv').attributes[1].value;
  console.log(a); //output: background-color: red; color: white;
  let b = document.getElementById('myDiv').attributes[0].value = "container";
  console.log(b); //output: container;
</script>
  5. removeAttribute: Removes the specified attribute from the element.

    6. Example

    <div id="myDiv" class="highlight">Hello, World!</div>
<script>
  document.getElementById('myDiv').removeAttribute('class');
</script>

DOM CSS styling Methods

  1. style: Provides access to inline styles of an element.

    2. Example

    <div id="myDiv" style="color: red; font-size: 18px;">Hello, World!</div>
<script>
	//Getting the value of the 'color' property
	document.getElementById('myDiv').style.color; // Output: "red"
	//Setting the value of the 'font-size'
	document.getElementById('myDiv').style.fontSize = "italic bold 20px Arial"; // Output: "italic bold 20px Arial"
	document.getElementById("myDiv").style.border = "1px solid red"; //Adding css property and value
</script>
  2. className: Gets or sets the value of the class attribute of an element.

    3. Example

    <div id="myDiv" class="highlight">Hello, World!</div>
<script>
  let divElement = document.getElementById('myDiv');
  console.log(divElement.className); // Output: "highlight"
  divElement.className = 'red-text'; // Change class name
</script>
  3. classList: Provides methods to add, remove, toggle, or check for the presence of classes on an element.

    4. Example

    <div id="myDiv" class="highlight">Hello, World!</div>
<script>
  let divElement = document.getElementById('myDiv');
  divElement.classList.add('bold'); // Add a class
  divElement.classList.remove('highlight'); // Remove a class
  console.log(divElement.classList.contains('bold')); // Output: true
	console.log(divElement.classList); //Output:  "bold xyz bold" (order matters)
</script>
