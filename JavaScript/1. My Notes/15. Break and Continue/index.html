JS Break & Continue

The break and the continue statements are the only JavaScript statements that can "jump out of" a code block.

  • The Break Statement
  • The Continue Statement
  • The label Statement

The Break Statement

The break statement is used to terminates or jump out of the current loops as well as the switch statement.

The syntax of the break statement is as follows:

break [label];
  • Here, the label is optional. The break statement, without a label reference, can only be used to jump out of a loop or a switch.
  • With a label reference, the break statement can be used to jump out of any code block.

Example 1: break in for loop

<script>
    for (let i = 0; i < 10; i++) {
        let text = (document.getElementById('forbreak-output-1'));
        if (i == 5) {
            text.innerHTML += `loop break at i = ${i} `;
            break;
        }
        text.innerHTML += (i) + '<br>';
    }
</script>

Example 2: break in while loop

<script>
    let count = 0;
    let text = document.getElementById('whilebreak-output-1');
    while (count < 10) {
        if (count == 5) {
            text.innerHTML += (`loop break at sum = ${count}`);
            break;
        }
        text.innerHTML += count + "<br>";
        count += 1;
    }
</script>

Example 3: break in switch statement

<script>
    let x = 2;
    switch (x) {
        case 1:
            document.getElementById('switchbreak-output-1').innerHTML = "x = 1, did not match";
            break;
        case 2:
            document.getElementById('switchbreak-output-1').innerHTML = "x = 2, matched. the break statement terminates the switch statement when x is equal to 2.";
            break;
        case 3:
            document.getElementById('switchbreak-output-1').innerHTML = "x = 1, did not match";
            break;
        default:
            document.getElementById('switchbreak-output-1').innerHTML = "Nothing Matched"
    }
</script>

The Continue Statement

The continue statement is used to skip the current iteration of the loop, if a specified condition occurs and jump to the next iteration.

The syntax of the continue statement is as follows:

continue [label];
  • Here, the label is optional. The continue statement (with or without a label reference) can only be used to skip one loop iteration.

Example 1: continue in for loop 

<script>
let hasCodeRun = false;
function forcont() {
    if (hasCodeRun) {
        return;
    }
    for (let i = 1; i < 20; i++) {
        if (i % 2 != 0) {
            continue;
        }
        document.getElementById('forcont-output-1').innerHTML += `${i}, `;
    }
    document.getElementById('forcont-output-1').innerHTML += `<br> The <b><code>continue</code></b> statement skip at odd numbers and prints the even numbers`;
    hasCodeRun = true;
}
</script>

Example 2: continue in while loop

<script>
    let i = 0;
    let text = document.getElementById('whcont-output-1');
    while (i < 20) {
        if (i % 2 === 0 && i % 4 === 0) {
            i++; // Increment 'i' to avoid an infinite loop
            continue; // Skips the current iteration when i divisible by both 2 and 4
        }
        text.innerHTML += (i) + ", ";
        i++;
    }
</script>

The label Statement

The label statement is used to create a label for a block of code. Labeled code represents a block of code where the label name is used to identify the block of code.

The syntax of the label statement is as follows:

label_name:
      // Single or Multiple Code
      // Statements

A label can be used to break or continue the block of code.

break label_name;
continue label_name;

Example 1: Using Labels with Nested Loops

<script>
    outerLoop: for (let i = 0; i < 3; i++) {
        innerLoop: for (j = 0; j < 3; j++) {
            if (i === 1 && j === 1) {
                break outerLoop;
            }
            document.getElementById('labelbreak-output-1').innerHTML += `i = ${i}, j = ${j} <br>`;
        }
    }
</script>

Example 2: Using Labels to Skip Iterations in a Nested Loop

<script>
    outerLoop: for (let i = 0; i < 3; i++) {
        innerLoop: for (j = 0; j < 3; j++) {
            if (i === 1 && j === 1) {
                continue outerLoop;
            }
            document.getElementById('labelcont-output-1').innerHTML += `i = ${i}, j = ${j} <br>`;
        }
    }
</script>
