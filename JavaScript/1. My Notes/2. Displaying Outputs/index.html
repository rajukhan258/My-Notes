JavaScript Output

JavaScript can "display" data in different ways:

  • Writing into an HTML element, using innerHTML.
  • Writing into the HTML output using document.write().
  • Writing into an alert box, using window.alert().
  • Writing into the browser console, using console.log().
  • Using window.print() method in the browser to print the content of the current window.

Using innerHTML

To access an HTML element, JavaScript can use the document.getElementById("id") method.

The id attribute defines the HTML element. The innerHTML property defines the HTML content:

Example:

Hi EveryOne.

<p id="demo">Hi EveryOne.</p>
<button type="button" class="btn btn-success mb-3" onclick="getElementById(`demo`).innerHTML = `<i><b><u>Hello World. </u></b></i>(content Changed)`">Click Here</button>

Using document.write()

For testing purposes, it is convenient to use document.write():

Example:

Using document.write() after an HTML document is loaded, will delete all existing HTML:

<button type="button" class="btn btn-success mb-3"onclick="document.write(`Hi, I am here because of document.write()`)">Click Here</button>></i>(content Changed)`">Click Here</button>

Using window.alert()

We can use an window.alert() to display data. We can skip the window keyword.

Example:

<button type="button" class="btn btn-success mb-3" onclick="alert(`Hi, I am here because of window.alert()`)">Click Here</button>

Using console.log()

For debugging purposes, We can call the console.log() method in the browser to display data.

Example:

Now click F12 on your keyboard to activate debugging. Then select Console in the debugger menu. Then click Run again.

<button type="button" class="btn btn-success mb-3" onclick="console.log(`Hi, I am here because of console.log()`)">Click Here</button>

Using Print()

JavaScript does not have any print object or print methods.

We cannot access output devices from JavaScript. The only exception is that you can call the window.print() method in the browser to print the content of the current window.

Example:

<button type="button" class="btn btn-success mb-3" onclick="window.print()">Window.print( )</button>
