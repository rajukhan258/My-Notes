Arrays

Array is a single variable that is used to store elements of different data types. An array allows you to store several values with the same name and access them by using their index number.

1. Declaration of an Array

There are basically two ways to declare an array.

  1. Using Array literals
  2. Using constructor

A. Using Array literals

Array literal is the simplest and the most common way to define an array in javascript. Array literal is a list of values enclosed in square brackets ([]).

Example:

let empty = [];  // defining empty array
let numbers = [1, 2, 3, 4, 5];  // defining array with integers
let strings = ["Hello", "World"];   // defining array with strings
let mixed = [1, "Hello", true, [1, 2, 3], {a: 1, b: 2}];  // defining array with mixed data

B. Using constructors

An array constructor is a function that takes a variable number of arguments and returns an array. The function is called with the keyword new and the array constructor name Array().

let arr = new Array();, this creates an array of length 0.

Example:

<script>
    
    let empty = new Array();    // array with 0 length
    let numbers = new Array(1, 2, 3, 4, 5);    // array with integer
    let mixed = new Array(1, "Hello", true, 2.5, [1, 2]);     // mixed array
    let arry = new Array(4);    //mixed array
    arry [0] = "Hello";
    arry[1] = false;
    arry[2] = null;
    arry[3] = undefined;
</script>

5. Accessing array elements

Using Index

We can access any element in an array using its index number. The index number starts from 0 and goes to the length of the array.

Example:

<script>
    let arr = [1, 10, "Hello", true, 2.5];
    // accessing first element
    document.getElementById("ary0").innerHTML = `arr[0] = ${arr[0]}`;
    // accessing third element
    document.getElementById("ary2").innerHTML = `arr[2] = ${arr[2]}`;
    // accessing last element
    document.getElementById("arylast").innerHTML = `arr[4] = ${arr[arr.length - 1]}`;
</script>


Using Loop

Example:

<script>
    function aryloop() {

6. Modifying Array elements

        
An array is mutable(modifiable). Once an array is created, we can modify the contents of
            the array elements.

        
To modify an array using index, we can directly assign a new value to the desired index.

        
Updating element


        

            
Example:

            
<script>
    const numbers = [4, 9, 7, 3, 8];
    numbers[0] = "Ram";
    numbers[2] = 10;
    numbers[(numbers.length)-1] = true;
    document.getElementById("updatetext").innerHTML = `updated arry = [${numbers}]`;
</script>

            
            

                

            

        



        



        
Deleting element

        
Please note that the `delete` operator in JavaScript is used to remove a property from an object. To
            remove an element from an array, it's better to use the `splice()` method. However, in this
            case, we are not removing any elements from the array, but rather modifying them.
        

        

            
Example:

            
<script>
    const numbers = [4, 9, 7, 3, 8];
    delete numbers[0]; //delete first element of the array
    delete numbers[2]; //delete third element of the array
    document.getElementById("deletedtext").innerHTML = `Updated array = [${numbers}]`;
</script>
