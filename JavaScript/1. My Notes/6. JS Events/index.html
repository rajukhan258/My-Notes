JS Events

Basic Events List

  • Click (onclick)
  • Double Click (ondblclick)
  • Right CLick (oncontextmenu)
  • Mouse Hover (onmouseenter)
  • Mouse Out (onmouseout)
  • Mouse Down (onmousedown)
  • Mouse Up (onmouseup)
  • Key Press (onkeypress)
  • Key Up (onkeyup)
  • Load (onload)
  • Unload (onunload)
  • Resize (onresize)
  • Scroll (onscroll)

Click (onclick)

Click button to see the result

Double Click (ondblclick)

Double click button to see the result

Right Click (oncontextmenu)

Right click on button to see the result

Mouse Hover (onmouseenter)

Hover mouse on button to see the result

Mouse Out (onmouseout)

Hover mouse on button and Remove to see the result

Mouse Down (onmousedown)

It works just like Onclick. Hover Mosue and press mouse keys on button to see the result

Mouse Up (onmouseup)

Hover Mouse on button, then click on it and release the mouse button to see the result

Key Press (onkeypress)

It works with form element or in body tag. click input box and press key

Key Up (onkeyup)

It works when you press and relese a key. click input box and press and release keys

Resize (onresize)

It's a windows Event and works only in body tag. It will be triggered if you resize the window

Scroll (onscroll)

It's a windows Event and works only in body tag. It will be triggered if you scroll the page.
