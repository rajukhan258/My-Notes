Date Object

Date and it's Mehods

JavaScript Date is an object that represents a date and time in the JavaScript programming language. It provides methods to fetch time, date, month, day, year and lots of other properties.

A JavaScript Date object is a point in time, represented as the number of milliseconds since midnight January 1, 1970 UTC.

JavaScript Date objects are always in local time even though they are stored in UTC.

  1. Create A Date Object
  2. Javascript Date Methods

1. Create A Date Object

The Date object in javascript is created using the Date() constructor with different arguments. It returns a Date object.

There are 4 different ways to create Date object in javascript.

  1. new Date()
  2. new Date(milliseconds)
  3. new Date(dateString)
  4. new Date() with Parameters

1. new Date()

The new Date() method creates a new Date object representing the current date and time.

Example

<script>
    // Creating a new Date object representing today's date
    let today = new Date();

    // Displaying the current date on the HTML element
    document.getElementById("newdate-output-1").innerHTML = today;
</script>

2. new Date(milliseconds)

new Date(milliseconds) method return date and time passed after given milliseconds elapsed from midnight 1 January 1970 UTC.

Example

<script>
    // 0 seconds after midnight 1 January 1970 UTC
    let date = new Date(0);
    document.getElementById("milliseconds-output-1").innerHTML = date;
    // 01 january, 1970 UTC + 100000000000milliseconds
    date = new Date(100000000000);
    document.getElementById("milliseconds-output-2").innerHTML = date;
</script>

3. new Date(dateString)

new Date(dateString) method takes a string argument and returns a Date object representing the date and time specified by the string.

Example

<script>
    // Creating date object from date string
    let date = new Date("05 05, 1996 17:00:00");
    document.getElementById("dateString-output-1").innerHTML = date;

    // another string format
    date = new Date("November 11, 1997 01:00:00")
    document.getElementById("dateString-output-2").innerHTML = date;
</script>

4. new Date() with Parameters

The new Date(year, month, day, hours, minutes, seconds, milliseconds) function in JavaScript takes up to seven parameters:

  • year and month are compulsory
  • while the others (day, hours, minutes, seconds, milliseconds) are optional
All parameters must follow the same order as listed.

Let's look at all the 7 parameters:

  • year:It is an integer value that represents the year. The value passed for the year should be full. example 2012 not 12. 2 digit year will be represented as 19xx
  • month:It is integer value from 0 to 11 representing January(0) to December(11)
  • day:It is an integer value representing a date. It should be from 1 to 31
  • hours:It is an integer that represents an hour on a 24-hour scale
  • minutes:It is an integer value that represents the minute
  • second:It is an integer value that represents second
  • milliseconds:It is an integer value that represents milliseconds

Example

<script>
    // using all parameters
    let date1 = new Date(2010, 6, 21, 9, 15, 55, 250)

    //// leaving milliseconds
    let date2 = new Date(2010, 6, 21, 9, 15, 55);

    // leaving seconds
    let date3 = new Date(2010, 6, 21, 9, 15);

    // leaving minutes
    let date4 = new Date(2010, 6, 21, 9);

    // leaving hours
    let date5 = new Date(2010, 6, 21);

    // only year and month
    let date6 = new Date(2010, 6);

    // if you think of removing month too
    // then year will be treated as milliseconds
    // remember this => new Date(milliseconds)

    document.getElementById("output-1").innerHTML = `${date1}<br>${date2} <br> ${date3} <br> ${date4} <br> ${date5} <br> ${date6}`;
</script>

NOTE: Year and Month are compulsary. If month is Removed too, year will be treated as milliseconds.
remember this => new Date(milliseconds)

2. Javascript Date Methods

  1. toDateString(): Returns the date portion of the Date object as a human-readable string.
  2. getFullYear(): Returns the year (four digits) for the specified date.
  3. getMonth(): Returns the month (0-11) for the specified date, where 0 is January and 11 is December.
  4. getDay(): Returns the day of the week (0-6) for the specified date, where 0 is Sunday and 6 is Saturday.
  5. getDate(): Returns the day of the month (1-31) for the specified date.
  6. getHours(): Returns the hour (0-23) for the specified date.
  7. getMinutes(): Returns the minutes (0-59) for the specified date.
  8. getSeconds(): Returns the milliseconds (0-999) for the specified date.
  9. getMilliseconds(): Returns the milliseconds (0-999) for the specified date.
  10. getTime(): returns the number of milliseconds since January 1, 1970, for a given Date object, indicating its position in time relative to the Unix Epoch.
  1. setFullYear(): Sets the year for the specified date.
  2. setMonth(): Sets the month (0-11) for the specified date.
  3. setDate(): Sets the day of the month (1-31) for the specified date.
  4. setHours(): Sets the hour (0-23) for the specified date.
  5. setMinutes(): Sets the minutes (0-59) for the specified date.
  6. setSeconds(): Sets the seconds (0-59) for the specified date.
  7. setMilliseconds(): Sets the milliseconds (0-999) for the specified date.
  8. setTime(): Sets the time for a Date object based on the number of milliseconds since January 1, 1970.
  9. Date.now(): Returns the number of milliseconds elapsed since January 1, 1970, 00:00:00 UTC for the current time.
  10. getTimezoneOffset(): Returns the time zone difference, in minutes, between the current local time and UTC (Coordinated Universal Time). Positive values indicate time zones west of UTC, and negative values indicate time zones east of UTC.

1. toDateString():

The toDateString() method in JavaScript is used to retrieve the date portion of a Date object as a human-readable string without including the time or timezone information.

It returns a string representation of the date in the format that depends on the implementation and the browser.

Example

<script>
    // Create a new Date object representing the current date and time
    const today = new Date();

    // Get the date portion as a string
    const dateStr = today.toDateString();
    document.getElementById("toDateString-output-1").innerHTML = dateStr;
</script>

2. getFullYear():

The getFullYear() method in JavaScript is used to retrieve the year (as a four-digit number) from a Date object.

Example

<script>
    const today = new Date();

    // Get the full year
    const year = today.getFullYear();
    document.getElementById("getfullyear-output-1").innerHTML = year;
</script>

3. getMonth():

The getMonth() method in JavaScript is used to retrieve the month from a Date object.

It returns the month as a zero-based index, where January is represented as 0, February as 1, and so on, until December, which is represented as 11.

Example

<script>
    const today = new Date();

    // Get the month
    const month = today.getMonth();
    document.getElementById("getmonth-output-1").innerHTML = month;
</script>

4. getDay():

The getDay() method in JavaScript retrieves the day of the week from a Date object as a zero-based index, where Sunday is represented as 0, Monday as 1, and so on, until Saturday, which is represented as 6.

Example

<script>
    const today = new Date();

    // Get the day of the week
    const day = today.getDay();

    // Output the day of the week
    document.getElementById("getday-output-1").innerHTML = day;
</script>

5. getDate():

The getDate() method in JavaScript is used to retrieve the day of the month (a number from 1 to 31) from a Date object.

It does not return the day of the week, but rather the numerical day within the month.

Example

<script>
    const today = new Date();

    // Get the day of the month
    const dayOfMonth = today.getDate();

</script>

19. Date.now()

Date.now() is a static method of the Date object in JavaScript, used to get the current timestamp in milliseconds since January 1, 1970, 00:00:00 UTC (Unix Epoch time).

Example

<script>

    const currentTime = Date.now();

    document.getElementById("datenow-output-1").innerHTML = currentTime;
</script>

20. getTimezoneOffset()

The getTimezoneOffset() method in JavaScript returns the time zone offset, in minutes, between the current local time and Coordinated Universal Time (UTC).

The getTimezoneOffset() method calculates the difference, in minutes, between the local time zone and UTC. It returns a positive value if the local time zone is behind UTC (west of UTC) and a negative value if it's ahead of UTC (east of UTC).

Example

<script>
    const today = new Date();   // Create a new Date object representing the current date and time

    // Get the time zone offset in minutes
    const offsetInMinutes = today.getTimezoneOffset();
    document.getElementById("zoneOffset-output-1").innerHTML = offsetInMinutes;
</script>
