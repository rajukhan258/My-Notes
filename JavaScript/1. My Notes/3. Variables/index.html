JS Variables

Variables are containers that can be used to store data like strings, numbers, objects, etc.

JavaScript Variables can be declared in 4 ways:

  • Using var
  • Using let
  • Using const
  • Automatically

All JavaScript variables must be identified with unique names. These unique names are called identifiers. Identifiers can be short names (like x and y) or more descriptive names (age, sum, totalVolume).

NOTE: JavaScript identifiers are case-sensitive.

Variable Naming

The general rules for constructing names for variables (unique identifiers) are:

  • Names can contain letters, digits, underscores, and dollar signs.
  • Names must begin with a letter
  • Since JavaScript treats underscore and dollar sign as a letter, identifiers begining with _ and $ are valid variable names
  • Names are case sensitive (y and Y are different variables)
  • Reserved words (like JavaScript keywords) cannot be used as names

Using Var [value can be redeclared]

Var variables can be re-declared and updated within its scope. This became increasingly difficult to debug.

Variables declared with var can have a global or local scope. Global scope is for variables declared outside functions, while local scope is for variables declared inside functions.

Example:

var x = 5; //Declaring Variable
var x = "Ram";  //Declaring Variable

Using let

The let keyword was introduced in ES6 (2015)

Variables defined with let can be updated but can't be Redeclared.

Variables defined with let must be Declared before use

Variables defined with let have Block Scope. It means variables declared inside a { } block cannot be accessed from outside the block:

Example:

let x = 5; //Declaring Variable
    x = "Ram"; //Updating Variable (Correct)

    let x = 5; //Declaring Variable
    let x = "Ram"; //Redeclaring Variable  (ERROR)

Using Const

The const keyword was introduced in ES6 (2015)

Variables defined with const can neither be updated nor Redeclared

Variables defined with const must be Declared before use

Variables defined with const have Block Scope. It means variables declared inside a { } block cannot be accessed from outside the block:

Example:

const PI = 3.14159265359;  //Correct

const PI;
PI = 3.14159265359;  //Incorrect

Const can be used to declare:

  • A new Array
  • A new Object
  • A new Function
  • A new RegExp

Constant Objects and Arrays

The keyword const is a little misleading. It does not define a constant value. It defines a constant reference to a value.

Because of this we can NOT:

  • Reassign a constant value
  • Reassign a constant array
  • Reassign a constant object

But we CAN:

  • Change the elements of constant array
  • Change the properties of constant object

Constant Arrays

We can change the elements of a constant array:

Example:

//We can create a constant array:
const Cars = ["Honda", "Volvo, "BMW"];

//But we can NOT Reassign the array directly:
Cars = ["Toyota", "Volvo", "Audi"];  // ERROR

//We can change an element:
Cars[0] = "Toyota";

//We can add an element:
Cars.push("Audi");

Constant Objects

We can change the properties of a constant object:

Example:

// We can create a const object:
const car = {type:"Fiat", model:"500", color:"white"};

//But you can NOT Reassign the object directly:
car = {type:"Volvo", model:"EX60", color:"red"};    // ERROR

// We can change a property:
car.color = "red";

//We can add a property:
car.owner = "Johnson";

Points to Remember:

  • let and const have block scope.
  • let and const can not be redeclared.
  • let and const must be declared before use.
  • let and const does not bind to this.
  • let and const are not hoisted.
  • var does not have to be declared.
  • var is hoisted.
  • var binds to this.

Frequently asked questions

1. What are variables used for in javascript programs?
Ans: Variables are used to store values in a program to be used later on.

2. How many types of variables are there in JavaScript?
Ans: In JavaScript, you can create 3 types of variables: using const , let , and var statements.

3. What is a global variable JavaScript?
Ans: Global variables are variables that are accessible in the entire program.

4. What is a local variable JavaScript?
Ans: Local variables are variables that are accessible only in the function where they are declared.
