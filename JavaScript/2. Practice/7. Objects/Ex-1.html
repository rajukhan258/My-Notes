Object Exercise

Exercises: Level 1

  1. Create an empty object called dog
  2. Print the the dog object on the console
  3. Add name, legs, color, age and bark properties for the dog object. The bark property is a method which return woof woof
  4. Get name, legs, color, age and bark value from the dog object
  5. Set new properties the dog object: breed, getDogInfo

Example

<script>
    // Create an empty object called dog
    const dog = {};

    // Print the dog object on the console
    console.log(dog);

    // Add name, legs, color, age, and bark properties for the dog object
    dog.name = "Tiger";
    dog.legs = 4;
    dog.age = 15;
    dog.color = "brown";
    dog.bark = function () {
        return "woof woof";
    };

    // Get name, legs, color, age, and bark value from the dog object
    const valueArray = [];
    for (let key in dog) {
        // Check if the property value is not a function
        if (typeof dog[key] !== 'function') {
            valueArray.push(dog[key]); // Push the property value
        } else {
            valueArray.push(dog[key]()); // Execute the function and push its result
        }
    }

    // Display name, legs, color, age, and bark values in a formatted way
    document.getElementById("output-1").innerHTML = `<b>Ans 4:</b> [${valueArray.join(", ")}]`;

    // Set new properties in the dog object: breed and getinfo
    dog.breed = "Labrador";
    dog.getinfo = function () {
        return `Name: ${this.name}<br> Age: ${this.age}<br> Color: ${this.color}<br> Breed: ${this.breed}`;
    };

    // Display the result of the getinfo method
    document.getElementById("output-2").innerHTML = `<b>Ans 5:</b> ${dog.getinfo()}`; 
</script>
