Program to Convert Kilometers to Miles

To convert miles to kilometers, you can use formula:

kilometers = miles / factor

<script>
    const km = prompt(`Enter value in Kilometers`);

    //1km = 0.621371 miles
    const factor = 0.621371;

    //calculate miles
    const miles = km * factor;

    document.getElementById("result").innerHTML = `${km} kilometers is equal to ${miles} miles.`;
</script>
