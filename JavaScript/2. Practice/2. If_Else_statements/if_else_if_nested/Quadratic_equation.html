Javascript Program to Solve Quadratic Equation

The standard form of a quadratic equation is:

    ax2 + bx + c = 0, where
    a, b and c are real numbers and
    a ≠ 0

To find the roots of such equation, we use the formula,

    (root1,root2) = (-b ± √b2-4ac)/2

The term b2-4ac is known as the discriminant of a quadratic equation. It tells the nature of the roots.

  • If the discriminant is greater than 0, the roots are real and different.
  • If the discriminant is equal to 0, the roots are real and equal.
  • If the discriminant is less than 0, the roots are complex and different.


<script>
    let root1, root2;

    // take input from the user
    let a = prompt(`Enter value of a`);
    document.getElementById("value-a").innerHTML = `Let a = ${a};`
    let b = prompt(`Enter value of b`);
    document.getElementById("value-b").innerHTML = `Let b = ${b};`
    let c = prompt(`Enter value for c`);
    document.getElementById("value-c").innerHTML = `Let c = ${c};`

    //calculate discriminant
    let discriminant = b * b - 4 * a * c;
    document.getElementById("discriminant").innerHTML = `discriminant = b2-4ac = ${discriminant} `;

    //condition for real and different roots
    if (discriminant > 0) {
        root1 = (-b + Math.sqrt(discriminant)) / (2 * a);
        root2 = (-b - Math.sqrt(discriminant)) / (2 * a);

        // result
        document.getElementById("result1").innerHTML = `The roots of the Quadratic equation are ${root1} and ${root2} `;
    }

    // condition for real and equal roots
    else if(discriminant == 0){
        root1 = root2 = -b / (2 * a);

        //result
        document.getElementById("result2").innerHTML = `The roots of the Quadratic equation are ${root1} and ${root2} `;
    }

    // if roots are not real
    else{
        let realPart = (-b / (2 * a)).toFixed(2);
        let imagPart = (Math.sqrt(-discriminant) / (2 * a)).toFixed(2);

        // result
        document.getElementById("result3").innerHTML = `The roots of quadratic equation are ${realPart} + ${imagPart}i and ${realPart} - ${imagPart}i`;
    }
    </script>
