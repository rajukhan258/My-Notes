Program to find largest Number

Given Numbers:


<script>
    const num1 = parseInt(prompt(`Enter num-1`));
    document.getElementById("n1").innerHTML = `${num1}`;
    const num2 = parseInt(prompt(`Enter num-2`));
    document.getElementById("n2").innerHTML = `${num2}`;
    const num3 = parseInt(prompt(`Enter num-3`));
    document.getElementById("n3").innerHTML = `${num3}`;

    //check the condition
    if(num1 >= num2 && num1 >= num3){
    document.getElementById("result").innerHTML=`The Largest number is ${num1}.`;
    }
    else if(num2 >= num1 && num2 >= num3){
    document.getElementById("result").innerHTML=`The Largest number is ${num2}.`;
    }
    else{
    document.getElementById("result").innerHTML=`The Largest number is ${num3}.`;
    }
</script>

Method - 2

JavaScript built-in Math.max() function to find the largest among the numbers.

        
<script>
    const num1 = parseInt(prompt(`Enter num-1`));
    document.getElementById("n1").innerHTML = `${num1}`;
    const num2 = parseInt(prompt(`Enter num-2`));
    document.getElementById("n2").innerHTML = `${num2}`;
    const num3 = parseInt(prompt(`Enter num-3`));
    document.getElementById("n3").innerHTML = `${num3}`;

    const largest = Math.max(num1, num2, num3);

    console.log('Largest Number:',largest);
</script>
