IF-Else tabular view

1. Input Name of student and marks in 3 subjects and calculate Total ,percentage and grade in a table
Grade- >=80 A
60-80 B
40-60 c
otherwise fail

Tabular Data

Student Name Maths Science Computer Total Percantage Grade 
            
<script>
    const na = prompt("Enter student name");
    const marks1 = parseFloat(prompt("Enter marks in Maths"));
    const marks2 = parseFloat(prompt("Enter marks in English"));
    const marks3 = parseFloat(prompt("Enter marks in Computer"));

    total = marks1 + marks2 + marks3;
    percent = (total / 300) * 100;

    document.getElementById("name").innerHTML = na;
    document.getElementById("sub1").innerHTML = marks1;
    document.getElementById("sub2").innerHTML = marks2;
    document.getElementById("sub3").innerHTML = marks3;
    document.getElementById("total").innerHTML = total;
    document.getElementById("percent").innerHTML = percent;
    // document.getElementById("grade").innerHTML = grade;

    if (percent >= 80) {
        document.getElementById("grade").innerHTML = ("A");
    }

    else if (percent <= 79 && percent >= 60) {
        document.getElementById("grade").innerHTML = ("B");
    }

    else if (percent <= 59 && percent >= 40) {
        document.getElementById("grade").innerHTML = ("C");
    }

    else if (percent <= 39) {
        document.getElementById("grade").innerHTML = ("Re-appear");
    }
</script>
