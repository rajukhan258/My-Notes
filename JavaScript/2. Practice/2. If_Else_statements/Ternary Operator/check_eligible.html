Ternary Operator

Checks if a person is older than 18 years or not and returns a message accordingly.

<script>
    let age = parseInt(prompt("Enter your Age:"));

    let canvote = (age >= 18) ? `Yes` : `No`;
    document.getElementById("output").innerHTML = `Age = ${age} <br/>Can Vote: <b>${canvote}</b>`;
</script>
