Ternary Operator without else

Example of a ternary operator without else.

<script>
    let price = parseInt(prompt("Enter the price of Product"));
    let discount = parseInt(prompt("Enter the Discount in numbers:"));

    let discountedprice = (discount >= 0) && (price - discount);
    document.getElementById("output").innerHTML = `Price = ${price} <br/> Discount = ${discount}<br/> Discounted Price = ${discountedprice}`
</script>
