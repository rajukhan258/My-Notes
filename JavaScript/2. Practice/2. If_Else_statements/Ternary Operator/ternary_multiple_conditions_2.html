Ternary operator multiple conditions

The ternary operator can also be used to check multiple conditions and assign a value to a variable based on that.

Example 2:

<script>
    let age = prompt("Enter your Age");
    let group = (age < 18) ? "Underage" : age < 65 ? "Adult" : "Senior Citizen";

    document.getElementById("output").innerHTML = `Age Group = <b>${group}</b>`

    //let group = age < 18 ? "underage"    // if

</script>
