Date Object Exercise

Q. Handle scenarios where you need to account for leap years in date calculations.

Example

<script>
    // Function to check if a year is a leap year
    function isLeapYear(year) {
        return (year % 4 === 0 && year % 100 !== 0) || (year % 400 === 0);
    }

    // Example usage of isLeapYear function
    const yearToCheck = 2024;
    const isYearLeap = isLeapYear(yearToCheck);

    document.getElementById("output-1").innerHTML = isYearLeap;
</script>
