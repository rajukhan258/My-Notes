Date Object Exercise

Q. What method do you use to get the day of the week (e.g., Monday, Tuesday) from a Date object?

Example

<script>
    // Create a new Date object
    const today = new Date();

    // Get the numeric representation of the current day of the week
    const day = today.getDay();

    // Array containing the names of the days of the week
    const dayOfWeek = ['Sunday', 'Monday', 'Tuesday', 'wednesday', 'Thrusday', 'Friday',
        'saturday'];

    // Get the string representation of the current day using the day index obtained earlier
    const DayString = dayOfWeek[day];

    // Display the current day of the week on an HTML element
    document.getElementById('output-1').innerHTML = `Today is ${DayString}.`;
</script>
