Arrays Exercise

Question 1

Declare an array with more than 5 number of elements. Find the length of your array. Get the first item, the middle item and the last item of the array

Example

<script>
    const webTechs = ['HTML', 'CSS', 'JavaScript', 'React', 'Redux', 'Node', 'MongoDB'];
    let LengthOfArray = webTechs.length;    // Get the length of the array
    const firstItem = webTechs[0];  // Get the first item (at index 0)

    // Get the middle item (if the array has an odd length)
    const middleItem = webTechs[Math.floor(LengthOfArray / 2)];
    const lastItem = webTechs[LengthOfArray - 1];   // Get the last item (at the last index)

    // Get a reference to an HTML element with the id "output-1" to display the output
    let displayOutput = document.getElementById("output-1");

    // Display the length of the array and the first, middle, and last items in the HTML element
    displayOutput.innerHTML += `Length of Array: ${LengthOfArray}<br> firstItem: "${firstItem}"<br> middleItem: "${middleItem}" <br> lastItem: "${lastItem}"`;
</script>
