Program to Check Armstrong Number Between Two Intervals

A positive integer is called an Armstrong number (of order n) if

abcd... = an + bn + cn + dn + ...

In the case of an Armstrong number of 3 digits, the sum of cubes of each digit is equal to the number itself. For example, 153 is an Armstrong number because

153 = 1*1*1+ 5*5*5 + 3*3*3

Similarly, 1634 is an Armstrong number because:

1634 = 1*1*1*1 + 6*6*6*6* + 3*3*3*3 + 4*4*4*4


<script>  
    const lowernum = parseInt(prompt("Enter a digit lower number"));
    const uppernum = parseInt(prompt("Enter a digit Upper number"));
    
    document.write(`ArmStrong numbers between ${lowernum} and ${uppernum}:  `)

    // looping through lowNumber to highNumber
    for (i = lowernum; i <= uppernum; i++) {
        const numberOfDigits = i.toString().length;
        let Sum = 0;

        // create a temporary variable
        let Temp = i;

        /* loop through a number to find if 
    a number is an Armstrong number */
        while (Temp > 0) {

            // finding the one's digit
            let Remain = Temp % 10;
            Sum += Remain ** numberOfDigits;

            // removing last digit from the number
            Temp = parseInt(Temp / 10);  // convert float into integer
        }
        // check the condition
        if (Sum == i) {
            document.write(`${i}, `)
        }
    }
</script>
