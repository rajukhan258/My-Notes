Sum of Natural Numbers

Using for Loop

The natural numbers include the positive integers (also known as non-negative integers) and a few examples include 1, 2, 3, 4, 5, 6, …∞. In other words, natural numbers are a set of all the whole numbers excluding 0.



<script>
    const number = parseInt(prompt(`Enter a positive number:`));
    let sum = 0;

    // looping from i = 1 to number
    for(i = 1; i <= number; i++){
        sum += i;
    }
    document.write(`The Sum is ${sum}`);
 </script>

Using While Loop



<script>
    const number = parseInt(prompt(`Enter a positive number:`));
    document.write(`Given Natural number: ${number} <br>`)
    let Sum = 0; i = 1;

    // looping from i = 1 to number
    while(i <= number){
        Sum += i;
        i++;
    }
    document.write(`The Sum of Natural numbers upto ${number} is ${Sum}.`);
</script>
