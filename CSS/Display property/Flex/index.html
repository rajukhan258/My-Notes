Display Flex Property


Flexbox can be divided into two parts:-
 1. Flex Container [parent element]
 2. Flex Items [Child elenents]

To start using the Flexbox model, you need to first define a flex container.

2
3

Code:-div.container{ background-color: dodgerblue; display: flex; }

div.container div{ background-color: #f1f1f1; margin: 10px; padding: 20px; font-size: 30px; }


1. Flex Container [Parent Element]


The flex container properties are:
 a. flex-direction
 b. flex-wrap
 c. flex-flow
 d. justify-content
 e. align-items
 f. align-content


a. flex-direction

Specifies the direction of the flexible items inside a flex container
 * row (default): left to right in ltr; right to left in rtl
 * row-reverse: right to left in ltr; left to right in rtl
 * column: same as row but top to bottom
 * column-reverse: same as row-reverse but bottom to top


flex-direction: row;

b. flex-wrap

Specifies whether the flex items should wrap or not, if there is not enough room for them on one flex line
 * nowrap (default): all flex items will be on one line
 * wrap: flex items will wrap onto multiple lines, from top to bottom.
 * wrap-reverse: flex items will wrap onto multiple lines from bottom to top.

d. justify-content

Horizontally aligns the flex items when the items do not use all available space on the main-axis
 * flex-start (default): aligns the flex items at the beginning of the container (this is default)
 * flex-end: value aligns the flex items at the end of the container
 * center: aligns the flex items at the center of the container:
 * space-between: items are evenly distributed in the line; first item is on the start line, last item on the end line
 * space-around: items are evenly distributed in the line with equal space around them.
 * space-evenly: items are distributed so that the spacing between any two items (and the space to the edges) is equal.

e. Align-items

Vertically aligns the flex items when the items do not use all available space on the cross-axis
 * stretch: stretches the flex items to fill the container (this is default)
 * flex-start: aligns the flex items at the top of the container
 * flex-end: aligns the flex items at the bottom of the container
 * center: aligns the flex items in the middle of the container
 * baseline: aligns the flex items such as their baselines aligns

f. Align-content

Modifies the behavior of the flex-wrap property. It is similar to align-items, but instead of aligning flex items, it aligns flex lines
 * stretch: stretches the flex lines to take up the remaining space (this is default)
 * flex-start: displays the flex lines at the start of the container
 * flex-end: displays the flex lines at the end of the container
 * center: displays display the flex lines in the middle of the container
 * space-around: displays the flex lines with space before, between, and after them.
 * space-between: displays the flex lines with equal space between them
 * space-evenly: items are evenly distributed with equal space around them


align-content: stretch; [default]

2. Flex Items [Child Element]


The flex container properties are:
 a. order
 b. flex-grow
 c. flex-shrink
 d. flex-basis
 e. flex [shorthand property]
 f. align-self


a. order

Specifies the order of the flex items inside the same container. The order value must be a number, default value is 0.

1
2
3

Code:-display: flex;
<div style="order: 3">1</div>
<div style="order: 1">2</div>
<div style="order: 2">3</div>


b. flex-grow

SSpecifies how much a flex item will grow relative to the rest of the flex items inside the same container. The value must be a number, default value is 0.


1
2
3

Code:-display: flex;
<div style="flex-grow: 1">1</div>
<div style="flex-grow: 1">2</div>
<div style="flex-grow: 5">3</div>


c. flex-shrink

specifies how much a flex item will shrink relative to the rest of the flex items. The value must be a number, default value is 1.


1
2
3
4
5
6
7
8

Code:-display: flex;
<div>1</div>
<div style="flex-shrink: 1">2</div>
<div>3</div>


d. flex-basic

specifies the initial length of a flex item.

1
2
3

Code:-display: flex;
<div>1</div>
<div style="flex-basis: 300px;">2</div>
<div>3</div>


e. flex

shorthand property for the flex-grow, flex-shrink, and flex-basis properties.
Make the second flex item not growable (0), not shrinkable (0), and with an initial length of 200 pixels:
The default is 0 1 auto


1
2
3
4
5

Code:-display: flex;
<div>1</div>
<div style="flex: 0 0 200px">2</div>
<div>3</div>


f. align-self

specifies the alignment for the selected item inside the flexible container.
The align-self property overrides the default alignment set by the container's align-items property.


1
2
3
4
5

Code:-display: flex;
<div>1</div>
<div style="align-self: center">2</div>
<div>3</div>


Other Values: flex-end, flex-star, auto
