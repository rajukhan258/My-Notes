Grid-items align

The align-items property aligns the grid items along the vertical axis.

  • align-items: Start/end/center/stretch(Default); //Used for Vertical Alignment
  • justify items: start/end/center/stretch(default); //used for Horizontal alignment
  • place-items:align-items justify-items; //shorthand
  • align-self: start/end/center/stretch(default); //used for V-alignment for specific grid
  • justify-self: start/end/center/stretch(default); //used for H-alignment for specific grid
  • place-self: align-self justify-self; //shorthand

Grid-track-align

The grid-track-align property aligns the grid tracks (rows and columns) in the grid container.

  • justify-content: start/end/center/stretch/space-around/space-between/space-evenly //used for H-alignment
  • align content: start/end/center/stretch/space-around/space-between/space-evenly //used for V-alignment
  • place-content:align-content justify content //shorthand
