Display properties

The display property specifies the display behavior (the type of rendering box) of an element. The default value in XML is inline, including SVG elements.

 6. Display: Grid [See seperate Grid page]

Displays an element as a block element. It starts on a new line, and takes up the whole width.

3. Display: flow-root

When using float property on a chile element in div the border of the div shifts up and seems the child has came out of the div, To stop this display: flow-root is used

CODE USED: 
            #box{
                border: 2px solid red;
                display: flow-root;
            }
            
            #child{
                background-color: green;
                width: 100px;
                height: 50px;
                float: left;
            }
