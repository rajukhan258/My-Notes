Simple Selectors

    Type Selector

        color: red;
    }

    Class Selector
    .head{
        color: red;
    }

    ID Selector
    #box{
        color: red;
    }

    
    Universal Selector
    *{
        color: red;
    }

    #box *{
        color: red;
    }

Combinator Selectors

    Descendant Selector
        p span{
            color: red;
        }

        .box p{
            color: red;
        }

        div p span{
            color: red;
        }

     Direct child selectors 
     p > span{
        color: red;
        }

     ul > li{
        color: red;
     }   

     #box > p{
        color: red;
     }

     Adjecent Sibling Selector
        p + span{
            color: red;
            }

        div + p{
            color: red;
        }    

        Ul + p{
            color: red;
        }

        #box + p{
            color: red;
        }

        General Sibling Selector  
        p ~ span{
            color: red;
            }

        div ~ p{
            color: red;
        }    

        ul ~ p{
            color: red;
        }

        #box ~ p{
            color: red;
            }

Attribute Selectors

  • tag[attr] - Element with specific Attirbute
  • tag[attr=val] - Attribute having exact value
  • tag[attrr^=val] - Attribute begins with value
  • tag[attr|=val] - Attribute begins with value OR is the first in a dash seperated list
  • tag[attr*=val] - Value occurs anywhere in attribute
  • tag[attr$=val] - Attribute ends with value
  • tag[attr~=val] - Value matches attribute in space seperated list
