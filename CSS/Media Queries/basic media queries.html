Media Queries

CSS Media Types

  • All
  • @import "print.css" print;
  • @media screen and (max-width:810px)
  • Speech
  • @media (prefer-color-scheme: Value) //Values: Light/Dark
  • @media (display-mode: Value) //Values: fullscreen/minimal-ui/browser
  • @media (orientation: Value) //Values: landscape/portrait
  • @page //Only margin and Size can be changed. It is used for printing pages
