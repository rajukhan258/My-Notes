Scroll Snapping

Section 1

The scroll-snap-type CSS property sets how strictly snap points are enforced on the scroll container in case there is one.


It can be used in whole page {html} or in Divs. The following are the properties and their values:


  1. scroll-snap-type
    • y mandatory
    • x mandatory
    • both mandatory
    • y proximity
    • x proximity

  2. scroll-snap-align
    • start
    • end
    • center
