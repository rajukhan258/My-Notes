3. Text Decoration

3.a) The text-decoration-line property is used to add a decoration line to text.

 Overline text decoration [text-decoration-line: overline;]

 Line-through text decoration [text-decoration-line: line-through;]

 Underline text decoration [text-decoration-line: underline;]

 Overline and underline text decoration. [ text-decoration-line: overline underline;]

 Note: It is not recommended to underline text that is not a link, as this often confuses the reader.

6. Text Shadow

 6a) The text-shadowproperty in CSS is used to add shadows to the text. The default value of the text-shadow property is none.


 Syntax:-  text-shadow: h-shadow v-shadow blur-radius color|none|initial| inherit;


 **Property values:

  *None: It represents no shadow added to the text, this is the default value.

     This is some text with No shadow. [text-shadow: none;]

  *H-shadow: This property is required & used to specify the position of horizontal shadow. It accepts the negative values.

  *V-shadow: This property is required & used to specify the position of vertical shadow. It also accepts the negative values.

     This is some text with shadow.[text-shadow: 2px 2px;]

  *Blur-radius: It is used to set the blur radius. Its default value is 0 & is optional.

     This is some text with shadow.[text-shadow: 2px 2px;]

  *Color: It is used to set the color of the shadow. It is optional.

     This is some text with shadow.[text-shadow: 2px 2px 5px red;]
