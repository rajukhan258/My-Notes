Bennett Feely website can be used to get codes and coodinates for different shapes. Click Here

Circle 
      #circle{
      width: 300px;
      border: 1px solid black;
      clip-path: circle(40% at 50% 50%);
      /* 40% radius,  50% x-asix,  50% y-axis */
    }
Ellipse 
      #ellipse{
      width: 300px;
      border: 1px solid black;
      clip-path: ellipse(10% 40% at 50% 50%);
      /* 10% vertical radius, 40% horizontal radius, 50% x-asix  50% y-axis */
    }
Inset 
      #inset{
      width: 300px;
      border: 1px solid black;
      clip-path: inset(25% 40% 34% 10%);
      /* top, right, bottom, left */
    }
Polygon 
      #polygon{
      width: 300px;
      border: 1px solid black;
      clip-path: polygon(50% 0, 0% 100%, 100% 100%);
      /* (x y, x y, x y) of P1, p2, p3 */
    }
rhombus 
       #rhombus{
      width: 300px;
      border: 1px solid black;
      clip-path: polygon(50% 0%, 100% 50%, 50% 100%, 0% 50%);
    }
