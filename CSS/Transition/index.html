Transition-Duration

Code Used #box3{
        border: 1px solid blue;
        width: 150px;
        height: 100px;
        background: pink;
        transition-property: width, height, background ;
        transition-duration: 4s, 2s;
        }
        #box3:hover{
        width: 300px;
        background: rgb(21, 79, 227);
        height: 110px;
        }
