CSS Font and Font Family

CSS Font property is used to control the look of texts. By the use of CSS font property you can change the text size, color, style and more.


These are some important font attributes:

1. Font color

There are three different formats to define a color:

 By a color name:-{ color: red;}

 By hexadecimal value:- { color: #9000A1;}

 By RGB:{color:rgb(0, 220, 98);}-



2. Font Family

1. Serif fonts have a small stroke at the edges of each letter. They create a sense of formality and elegance.

 a).Times New Roman: This is a paragraph, shown in the Times New Roman font.
 {font-family: "Times New Roman", serif;}

 b).Georgia: This is a paragraph, shown in the Georgia font.
 {font-family: "Georgia", serif;}

 c). Garamond: This is a paragraph, shown in the Garamond font.
 {font-family: "Georgia", serif;}

 Other Fonts:- Big Calson, Book Antique, Bookman, cambria, Didot, palatino, Rockwell, Baskerville.


2. Sans-serif fonts have clean lines (no small strokes attached). They create a modern and minimalistic look.

 a). Arial: This is a paragraph, shown in the Arial font.
 {font-family: "Arial", Sans-serif;}

 b). Verdana: This is a paragraph, shown in the Verdana font.
 {font-family: "Verdana", Sans-serif;}

 c). Helvetica: This is a paragraph, shown in the Helvetica font.
 {font-family: "Helvetica", Sans-serif;}

 Other Fonts:- calibri, Noto, GIll Sans, Candara, Futara, Geneva, Optima, Avanta Garde.


3. Monospace fonts - here all the letters have the same fixed width. They create a mechanical look.

 a). Courier New:This is a paragraph, shown in the Courier New font.
 {font-family: "Courier New", Monospace;}

 b). Lucida Console: This is a paragraph, shown in the Lucida Console font.
 {font-family: "Lucida Console", Monospace;}

  c). Monaco: This is a paragraph, shown in the Monaco font.
  {font-family: "Monaco", Monospace;}

 Other Fonts:- Consolas, Courier, Andale Mono, Lucidatypewritter


4. Cursive fonts imitate human handwriting.

 a). Brush Script MT:This is a paragraph, shown in the Brush Script MT font.
 {font-family: "Brush Script MT", Cursive;}

 b). Lucida Handwriting: This is a paragraph, shown in the Lucida Handwriting font.
 {font-family: "Lucida Handwriting", Cursive;}

 Other Fonts:- Comic Sans, Comic Sans MS, Parkavenue, Florence, Coronet script, Bradley Hand.


5. Fantasy fonts are decorative/playful fonts.

 a). Copperplate:This is a paragraph, shown in the Copperplate font.
 {font-family: "Copperplate", Fantasy;}

 b). Papyrus: This is a paragraph, shown in the Papyrus font.
 {font-family: "Papyrus", Fantasy;}

 Other Fonts:- Impact, Luminari, Jazz LET, Blippo, Oldtown, Trattatello, Market Felt.



3. Font Size

CSS font size property is used to change the size of the font.

  {font-size: xx-small;} = This font size is extremely small.

  {font-size: x-small;} = This font size is extra small

  {font-size: small;} = This font size is small

 {font-size: smaller;} = This font size is smaller.

  {font-size: medium;} = This font size is medium.

  {font-size: large;} = This font size is large.

 { font-size: larger;} = This font size is larger.

 { font-size: x-large} = This font size is extra large.

 {font-size: xx-large;} = This font size is extremely large.

 {font-size: 200%;} = This font size is set on 200%.

 { font-size: 20px;} = This font size is 20 pixels.

4. Font Style

CSS Font style property defines what type of font you want to display. It may be italic, oblique, or normal.

 a). {font-style: normal;} = This is a paragraph in normal style.

 b).{font-style: italic;} = This is a paragraph in italic style.

 c).{font-style: oblique;} = This is a paragraph in oblique style.

 d). {font-style: oblique 20deg;} = This is a paragraph in oblique style at degree 20.



5. Font Variant

CSS font variant property specifies how to set font variant of an element. It may be normal and small-caps.

 {font-variant: normal; } = This paragraph is shown in normal font.

 {font-variant: small-caps;} = This paragraph is shown in small-caps.



6. Font Weight

The possible values of font weight may be normal, bold, bolder, lighter or number (100, 200..... upto 900).

 {font-weight:bold;} = This font is bold.

 {font-weight:bolder;} = This font is bolder.

 {font-weight:lighter;} = This font is lighter.

 {font-weight:100;} = This font weight is 100.

 {font-weight:200;} = This font weight is 200.

 {font-weight:300;} = This font weight is 300.

 {font-weight:400;} = This font weight is 400.

 {font-weight:500;} = This font weight is 500.

 {font-weight:600;} = This font weight is 600.

 {font-weight:700;} = This font weight is 700.

 {font-weight:800;} = This font weight is 800.

 {font-weight:900;} = This font weight is 900.



7. Font Stretch

The font-stretch property allows you to make text narrower (condensed) or wider (expanded).

Note: Some fonts provide additional faces; condensed faces and expanded faces. For these fonts, you can use the font-stretch property to select a normal, condensed, or expanded font face.

Note: This property has no effect if the selected font does not offer condensed or expanded faces!


 a). {font-stretch: ultra-condensed;} Makes the text as narrow as it gets

 b). {font-stretch: extra-condensed;} Makes the text narrower than condensed, but not as narrow as ultra-condensed

 c). {font-stretch: condensed;} Makes the text narrower than semi-condensed, but not as narrow as extra-condensed

 d). {font-stretch: semi-condensed;} Makes the text narrower than normal, but not as narrow as condensed

 e). {font-stretch: normal;} Default value. No font stretching

 f). {font-stretch: semi-expanded;} Makes the text wider than normal, but not as wide as expanded

 g). {font-stretch: expanded;} Makes the text wider than semi-expanded, but not as wide as extra-expanded

 h).{font-stretch: extra-expanded;} Makes the text wider than expanded, but not as wide as ultra-expanded

 i). {font-stretch: ultra-expanded;} Makes the text as wide as it gets



8. Font Shorthand

The font property is a shorthand property for:
 font-style
 font-variant
 font-weight
 font-size/line-height
 font-family

Note: The font-size and font-family values are required. If one of the other values is missing, their default value are used.

  {font: 20px Arial, sans-serif;}
  This is a paragraph. The font size is set to 20 pixels, and the font family is Arial.

  {font: italic bold 12px/30px Georgia, serif;}
  This is a paragraph. The font is set to italic and bold, the font size is set to 12 pixels, the line height is set to 30 pixels, and the font family is Georgia.
