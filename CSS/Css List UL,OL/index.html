CSS List UL / OL

In HTML, there are two main types of lists:

  1. unordered lists (<Ul>) - the list items are marked with bullets
  2. ordered lists (<Ol>) - the list items are marked with numbers or letters.

The CSS list properties allow you to:

  • list-style-type: This property is responsible for controlling the appearance and shape of the marker.
  • list-style-image: It sets an image for the marker instead of the number or a bullet point.
  • list-style-position: It specifies the position of the marker.
  • list-style: It is the shorthand property of the above properties.


List-Style-Type


The list-style-type property specifies the type of list item marker.


For Ordered List

  1. list-style-type:decimal;(default)
  2. list-style-type:lower-alpha;
  3. list-style-type:upper-alpha;
  4. list-style-type:lower-roman;
  5. list-style-type:upper-roman;
  6. list-style-type:string; (used here in CSS)
  7. list-style-type:none;

For Unordered List

  • list-style-type:disc;(default)
  • list-style-type:circle;
  • list-style-type:square;
  • list-style-type:none;

NOTE: Pseudo Element ::marker also can be used to change list style type



List-Style-Position


The list-style-position property specifies the position of the list-item markers (bullet points).


  • list-style-position:outside; (default)
  • list-style-position:inside;


List-Style-image


The list-style-image property specifies an image as the list item marker:


  • list-style-image: none;
  • list-style-image: url("starsolid.gif");
  • list-style-image: linear-gradient(to left bottom, red, blue);


List-Style (Shorthand)


The list-style property is a shorthand property. It is used to set all the list properties in one declaration. When using the shorthand property, the order of the property values are:

  • list-style-position:
