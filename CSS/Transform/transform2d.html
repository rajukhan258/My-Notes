Transform 2D

Transform 2D Values

  • None (default)
  • rotate(angle)
  • translate(x,y)
  • translate(x)
  • translate(y)
  • scale(x,y)
  • scale(x)
  • scale(y)
  • skew(x-angle, y-angle)
  • skewX(angle)
  • skewY(angle)
  • matrix(n,n,n,n,n)

Transform Origin and values

Transform-origin: X Y

  • left top
  • left bottom
  • right top
  • right top
  • 20% 30%

rotate(angle)

Hello
#rotate{
        width: 150px;
        height: 100px;
        background: lightseagreen;
        margin-left: 100px;
        transform-origin: right top;
    }
    #rotate:hover{
        transform: rotate(30deg); // values can be Negative
    }
