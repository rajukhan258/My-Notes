Transform 3D

Transform 3D Values

  • None (default)
  • perspective(px)
  • rotateX(angle)
  • rotateY(angle)
  • rotateZ(angle)
  • rotate3d(x,y,z,angle)
  • translateZ(z)
  • translate3d(x,y,z)
  • scale3d(x,y,z)
  • scaleZ(z)
  • matrix(n,n,n,n,n,n,n,n,n,n,n,n,n,n,n,n)

rotateX(angle)

Hello
CODE USED: #rotatex{
            width: 150px;
            height: 125px;
            background: rgb(234, 104, 104);
            transform: rotateX(50deg);
            float: left;
        }

perspective

Hello
CODE USED: #perspective{
            width: 150px;
            height: 125px;
            background: rgb(234, 104, 104);
            transform-origin: 50% 50%;
            transform: perspective(800px) rotateX(50deg);
            float: left;
        }
Prespective values: 
    
            
  • 50% 50%
    • 
            
  • left top (0% 0%)
    • 
            
  • top (50% 0%)
    • 
            
  • top right(0% 50%)
    • 
            
  • right(50% 100%)
    • 
            
  • bottom right(100% 100%)
    • 
            
  • bottom(50% 100%)
    • 
            
  • left bottom(0% 100%)
    • 
            
  • left(0% 50%)
    •
