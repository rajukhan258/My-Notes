Border-image(Shorthand sequence)

border-image: source | slice | width | outset | repeat;

The border-image property is a shorthand for the border-image-source, border-image-slice, border-image-width, border-image-outset, and border-image-repeat properties.

The border-image-source property specifies the image to be used as the border image.
