The border-style Property

1. Border style

border-style: dotted

border-style: dashed;

border-style: solid;

border-style: double;

border-style: groove;

border-style: ridge;

border-style: inset;

border-style: outset;

border-style: hidden;

border-style: None;

border-style: dotted dashed solid double;

Individual border

border-top-style: dotted; border-right-style: solid; border-bottom-style: dashed; border-left-style: ridge;

The border-style property can have from one to four values

One value

border-style: dotted; /* all sides dotted */

Two value

border-style: dotted solid; /* dotted top and bottom, solid left and right */

Three value

border-style: dotted solid double; /* dotted at top, solid on the sides, double at bottom*/

Four value

border-style: dotted solid double dashed; /* dotted at top, solid at right, double at bottom and dashed at left */



2. Border width

border-width:5px

border-width:medium

border-width:2px

border-width:thick

border-width:15px

Note: The "border-width" property does not work if it is used alone. Always specify the "border-style" property to set the borders first.

Specific side width

The border-width property can have from one to four values (for the top border, right border, bottom border, and the left border):

One value

border-width: 5px; /* 5px top and bottom left and right */

Two value

border-width: 5px 20px; /* 5px top and bottom, 20px left and right */

Three value

border-width: 20px 5px 30px; /* 20px top, 5px on the sides, 30px bottom*/

Four value

border-width: 25px 10px 4px 35px; /* 25px top, 10px right, 4px bottom and 35px left */



3. Border color

Note: If border-color is not set, it inherits the color of the element.

A solid Red border: {border-style:solid; border-color: red;}

A dahsed green border: {border-style:dashed; border-color: green;}

A dotted blue border: {border-style:dotted; border-color: blue;}

Note: The "border-color" property does not work if it is used alone. Use the "border-style" property to set the borders first.

Specific side color

The border-color property can have from one to four values (for the top border, right border, bottom border, and the left border):

One value

border-color: blue; /* blue all sides */

Two value

border-color: red blue; /* red top and bottom, blue right and left */

Three value

border-color: red blue green; /* red top, green bottom, blue right and left */

Four value

border-color: red green blue yellow; /* red top, green right, blue bottom and yellow left */

The color of the border can also be specified using a
hexadecimal value(HEX):- border-color: #bbbbbb; /* grey */
RGB values:- border-color: rgb(255, 0, 0); /* red */
HSL values:- border-color: hsl(0, 100%, 50%); /* red */



3. Border Raduius

This property is used to add rounded borders to an element:

border-radius: 5px;

border-radius: 10px;

border-radius: 20px;

Note: The "border-radius" property does not work if it is used alone. Use the "border-style" property to set the borders first.

Specific side Radius

The border-radius property can have from one to four values (border-top-left-radius, border-bottom-right-radius, border-bottom-left-radius, border-top-right-radius):

One value

border-radius:15px; /*radius applied on all four corners*/

Two value

border-radius:5px 15px; /* 5px top-left and bottom-right, 15px bottom-left and top-right, */

Three value

border-radius:5px 15px 20px; /* 5px top-left, 15px bottom-left and top-right, 20px bottom-right*/

Four value

border-radius:5px 10px 15px 20px; /* 5px top-left, 10px top-right, 15px bottom-right, 20px bottom-left*/



4. Border Shorthand property

The border property is a shorthand property for the following individual border properties:
  border-width
  border-style (required)
  border-color

border: 3px solid red; /* border-width, border-style, and border-color */

The border-left shorthand Property

This property is a shorthand property for border-left-width, border-left-style, and border-left-color.

border-left: 6px solid red;

The border-right shorthand Property

This property is a shorthand property for border-right-width, border-right-style, and border-right-color.

border-right: 6px solid red;

The border-top shorthand Property

This property is a shorthand property for border-top-width, border-top-style, and border-top-color.

border-top: 6px solid red;

The border-bottom shorthand Property

This property is a shorthand property for border-bottom-width, border-bottom-style, and border-bottom-color.

border-bottom: 6px solid red;
