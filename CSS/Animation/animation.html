animation-name and Animation-duration

}

animation-timing-function

Values
  • animation-name
  • animation-duration
  • animation-iteration-count
  • animation-direction
Unit Descrition
Ease Slow start, then fast, then end slowly
Linear Same speed from Start to End
Ease-in Slow start
Ease-out slow end
Ease-in-out slow start and end
step-start Equivalent to steps (1, start)
Step-end Equivalent to steps (1, end)
steps(#, start/end)
Cubic-bezier Lets you define your own Values(0 to 1)

animation(shorthand)

sequence
  • animation-name
  • animation-duration
  • animation-timing-function
  • animation-delay
  • animation-iteration-count
  • animation-direction
Hello
CODE USED.card5{
    width: 150px;
    height: 150px;
    margin-left: 30%;
    line-height: 150px;
    background-color: white;
    color: #03446A;
    text-align: center;
    font-size: 40px;
    box-shadow: 0 0 15px rgba(0, 0, 0, 0.1);
    animation: sample cubic-bezier(0.075, 0.82, 0.165, 1) 2s infinite alternate;
}
@keyframes sample{
    0%{border-radius: 0%;}
    100%{border-radius: 50%; background-color: tomato; color: white;}
}
